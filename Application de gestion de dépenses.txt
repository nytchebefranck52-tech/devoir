   Description du projet

GOOD GESTION est une application cliente–serveur développée en Python, permettant à un utilisateur de gérer son budget mensuel et ses dépenses personnelles de manière sécurisée.

L’application repose sur :

-une communication réseau UDP

-un chiffrement symétrique (Fernet / AES)

-une interface graphique (wxPython)

-une visualisation graphique du budget (matplotlib)

-Chaque utilisateur est identifié par un mot de passe, utilisé comme clé de chiffrement pour protéger les données échangées.

   Architecture générale

L’application suit une architecture client / serveur :

[ Client wxPython ]  <--UDP chiffré-->  [ Serveur Python ]


-Le serveur centralise les données (profil, budget, dépenses)

-Le client fournit l’interface graphique et les interactions utilisateur

-Toutes les données échangées sont chiffrées

   Sécurité et chiffrement

Le projet utilise :

-SHA-256 pour dériver une clé à partir du mot de passe

-Fernet (cryptography) pour le chiffrement symétrique

-pickle pour la sérialisation des objets Python

Principe :

-Le mot de passe est transformé en clé cryptographique

-Les données sont sérialisées

-Les messages sont chiffrés avant l’envoi

-Le serveur renvoie uniquement des données chiffrées
